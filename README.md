# Capstone 3: Power BI Sales Analysis

This project analyzes EmporiUm sales data using Power BI. The report shows sales trends over time, sales by category, sales by state, and top-selling books with author names for my assigned region which is west.

The goal of this project is to help the regional sales team understand performance and identify useful insights for business decisions.

## Tools Used

* Power BI
* Power Query
* GitHub

## Report Includes

* Sales trend line chart
* Sales by category chart
* Sales by state donut chart
* Top-selling books table with authors
